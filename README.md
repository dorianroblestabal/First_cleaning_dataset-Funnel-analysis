# First_cleaning_dataset-Funnel-analysis
# 🧹 Data Cleaning Project

## 📌 Overview
This project focuses on cleaning and preparing a raw dataset for further analysis.

The dataset contained multiple data quality issues that required transformation to make it usable.

---

## 🎯 Objectives
- Identify and fix data quality issues
- Standardize formats and data types
- Handle missing and inconsistent values
- Prepare the dataset for analysis

---

## 🧠 Key Tasks

### 🔹 Data Inspection
- Explored dataset structure
- Identified missing values and inconsistencies

### 🔹 Data Cleaning
- Handled null values
- Fixed data types
- Standardized formats (strings, dates, etc.)
- Removed or treated duplicates

### 🔹 Data Transformation
- Created new variables where needed
- Cleaned key columns for consistency

---

## 🛠️ Tools
- PostgreSQL

---

## 🚀 Outcome
The dataset was successfully cleaned and prepared for future analysis.
